# Интернет магазин технотоваров
## Карточка товара с использованием SASS

### Реализованные элементы:

#### 1. Переменные
- `$primary-color` - основной цвет (#3498db)
- `$secondary-color` - вторичный цвет (#2ecc71)
- `$text-color` - цвет текста
- `$background-color` - цвет фона
- `$border-radius` - радиус скругления
- и другие

#### 2. Миксины
- `@mixin card-shadow()` - создание тени с эффектом hover
- `@mixin button-style()` - стилизация кнопок с параметрами

#### 3. Вложенная структура-0gi
Использована методология БЭМ с вложенными селекторами:

```scss
.product-card {
    &__image { ... }
    &__content { ... }
    &__title { ... }
    // и т.д.
}
