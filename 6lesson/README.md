# TechShop

## Практика 1
- Создана стилизованная карточка товара. В коде применены **переменные** (для цветовой схемы), **миксин** (для универсального оформления кнопки).

## Практика 2
- Разработан сервер на **Express.js**, который обрабатывает запросы к коллекции товаров. Реализованы все пять стандартных эндпоинтов (создание, чтение всех товаров, чтение одного товара по id, редактирование и удаление).

## Практика 3
- С помощью **Postman** было выполнено тестирование созданного ранее API: отправлены запросы на добавление, получение и удаление товаров. Также были отправлены учебные запросы к внешним API.

## Практика 4
- С помощью React написан фронтенд для сайта, который запущен на другом порте. Полностью работоспособен и обращается к бэкенду по средствам REST API.

## Практика 5
- В серверное приложение интегрирована автоматическая генерация документации. С помощью библиотек **swagger-jsdoc** и **swagger-ui-express** был создан интерактивный интерфейс по адресу `/api-docs`, в котором описаны все эндпоинты, структуры запросов и ответов, а также доступна функция тестирования.
