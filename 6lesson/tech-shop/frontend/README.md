# Tech Shop Frontend

Фронтенд-приложение интернет-магазина Tech Shop.

Приложение взаимодействует с backend API и позволяет просматривать и управлять товарами.

##  Технологии

- React
- Vite
- JavaScript
- Fetch API

##  Реализованная функциональность

- Отображение списка товаров
- Добавление нового товара
- Редактирование товара
- Удаление товара
- Получение данных с backend API
- Обработка ошибок

Приложение взаимодействует с backend по адресу:

http://localhost:3000/api/products

## ▶ Запуск проекта

1. Установить зависимости:

npm install

2. Запустить приложение:

npm run dev

После запуска приложение будет доступно по адресу:

http://localhost:5173

Перед запуском фронтенда необходимо запустить backend-сервер.
